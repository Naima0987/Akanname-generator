# Akan Name Generator

## Author

Naima Abdullahi

---

## Description

The Akan Name Generator is a simple web application that generates a traditional Akan name based on a user's date of birth and gender. Akan names originate from Ghanaian culture, where children are named according to the day of the week they are born.

This project demonstrates basic front-end development skills using HTML, CSS, and JavaScript, including form handling, event listeners, and dynamic DOM manipulation.

---

## Setup Instructions

To run this project locally:

1. Clone the repository:

   ```
   git clone https://github.com/Naima0987/Akanname-generator.git
   ```

2. Navigate into the project folder:

   ```
   cd akan-name-generator
   ```

3. Open the project:

   * Double-click the `index.html` file
     **OR**
   * Open it using VS Code and run with Live Server
